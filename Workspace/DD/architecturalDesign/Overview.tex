This chapter describes the software system, the relationships between software components and the interaction between the actors and the system. It also describes resources, that are all the elements used by the external component to design such a services. To do so this section is divided in three different parties:
\begin{itemize}
	\itemBold{Component View} Describe the division of the system in layers and how the work flow is organized through them. It contains the UML component diagram and fast definition of all the parties
	\itemBold{Deployment View} Explains how the division of the system in Tiers is done.
	\itemBold{Runtime View} Describes with some UML Sequence diagram how each software component interacts with each other. 
\end{itemize}
